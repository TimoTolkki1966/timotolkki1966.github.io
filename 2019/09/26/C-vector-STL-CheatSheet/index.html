C++ vector STL CheatSheet

2019-09-26

markdown cheatsheet

One thing confusing about vector (and other container in C++) is that you do not know wether you should use the iterator or integer as the argument.

For example:

The v.at() takes a (normally) integer indicating the index of the postion, while v.erase() takes an iterator.

I think there is a (hidden) rule that makes it easy for the developer to tell what type they should send to the function. But i have not found that rule yet.

Anyway, the cheatsheet:

reference at (size_type n);

myvector.at(i)=i;
