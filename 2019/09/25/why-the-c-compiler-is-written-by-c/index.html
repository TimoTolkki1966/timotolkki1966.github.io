Why the C compiler is written by C

2019-09-25

Read this on an article about self-reproducing c programs:

The C compiler is written in C.
– Reflections on Trusting Trust

It sounds like a chicken-and-egg paradox. But actually this is simple: you write the most basic parts of C by Assembly, then you use them to build the rest of C. This is known as bootstrapping (wiki)). As they will all envetually be interpreted as Assembly, it makes no difference to the computer (but make human’s work much easier).

The Golang actually does the same. The basic of Golang is written by C and compiled by gcc. Then they use the basic Golang to implement the Golang compiler (gccgo). You can find gccgo information on the Golang github page here.
