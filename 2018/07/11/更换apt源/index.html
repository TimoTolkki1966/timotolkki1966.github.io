更换apt源

2018-07-11

(解决update时aliyun err问题)

之前apt好像一直默认的都是阿里云的源，最近apt update的时候发现慢的不行而且报错（校园网的锅？），于是换成了清华的源。只需要该一下etc/apt下的sources.list文件就好了。

首先进入到apt的路径

cd /etc/apt

然后备份一下原文件：

sudo mv sources.list sources.list_backup

新建一个和原来同名的文件

sudo vim sources.list

sudo apt update
sudo pat upgrade
